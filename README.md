🏠  Interior Design Management System

📌 Project Description

The Interior Design Management System is a web-based application developed using JSP, Servlets, Java, HTML, CSS, and MySQL.
This system allows users to browse interior design options, select designs, book services, make payments, and provide feedback.
Unlike many systems, this project allows users to directly enter their details while booking, so no login or registration is required.

🎨 Design Selection

Users can explore and select interior designs based on different categories:
- Residential Designs
- Commercial Designs
- Educational Designs

Designs can also be filtered by:
- Category
- Style
- Price Range
Users can add multiple designs to the cart before booking.

🛒 Booking System

Users can book designs by entering their details directly in the booking form.
Information collected:
- Full Name
- Email
- Phone Number
- Address
- Appointment Date
- Time Slot
- Special Requirements

After submitting the form, the system stores the booking in the database and displays a Booking Successful message.

💳 Payment System

After confirming a booking, users can proceed with payment.
The system records:
- Payment details
- Amount paid
- Payment status
Once payment is completed, a Payment Successful message is displayed.

💬 Feedback System

The system allows users to submit feedback about the service.
Features:
- Any visitor can submit feedback
- Both booked and non-booked users can provide feedback
- Feedback messages are stored in the database

📊 Admin Dashboard

The Admin Dashboard helps the administrator monitor the system.
Admin can view:-
- Total Bookings
- Total Revenue
- Total Designs
- Total Payments
- Highest Selling Design with details
- Search functionality
- Booking and payment status
  
The dashboard also displays database tables:
- Booking Records
- Payment Records
- Feedback Records

🛠 Technologies Used
- Java

- Servlets
- HTML
- CSS
- MySQL
- Apache Tomcat
- NetBeans IDE
